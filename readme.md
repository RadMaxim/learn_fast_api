<h3> docker build -t my_first_image .</h3>
<p> нужна для создания Docker-образа из Dockerfile.</p>
<h3>docker run --name=my_first_container -p 1252:8000 my_first_image</h3>
<p>нужна для запуска контейнера из Docker-образа.</p>
<h3>docker exec -it cont1 bash</h3>
<p>нужна для входа внутрь уже запущенного Docker-контейнера.</p>
